# Flutter UI Botanicaly

Flutter ui Botanicaly is easiest way to create new flutter project with clean code and well organized file folder.

## Features :

- Light & Dark Themes
- Common UI Screen (Login, Register, Dashboard, Detail)
- Material Design
- Loading Screen
- Media Query
